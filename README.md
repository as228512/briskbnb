# Briskbnb

[Briskbnb Live][heroku]

[heroku]: https://briskbnb.herokuapp.com

Briskbnb is an Airbnb inspired, home rental full-stack application, with a cold climate theme twist.

- Front-end: React.js + Redux
- Database: PostgreSQL
- Back-end: Ruby on Rails
